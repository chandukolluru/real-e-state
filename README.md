# Hyderabad Realty - Real Estate Website

## How to Run

1. **Install dependencies:**
   ```
   npm install express cors
   ```
2. **Start the backend server:**
   ```
   node server.js
   ```
3. **Open `index.html` in your browser** (or use a static server for the frontend).
